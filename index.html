Behaviors Library

The Behaviors Library is a collection of code bundles that can be easily attached to corona display objects to give them specific game and app behaviors.

-- Load two behaviors
local randMover = require "behaviors.dummy.randMover"
local randColor = require "behaviors.dummy.randColor"

-- Create an object.
local obj = newRect( layers.content, centerX-100, centerY, {} )

-- Create a list of behaviors to attach (optionally initializing behaviors with custom settings)
local behaviorsList = { 
    randColor( { colors = { _R_, _G_, _B_, _Y_  } } ),
    randMover( { delta = 10 } ),    
}

-- Attach the behaviors to the objects.. and done!
behaviors.attach( obj, behaviorsList )

Note: While the behaviors themselves are free, but they require the Behaviors Manager Plugin(#plugins) to work.

Templates

Templates are "complete games and applications," designed for you to learn from and to create with these primary goals:

  1. Provide a starting point for you to make your own games and apps by reskinning.
  2. Provide a starting point for you to expand and extend the game/app concept.
  3. Keep game and app development simple and fun.
