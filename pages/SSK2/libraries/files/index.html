Files Library (ssk.files.*)

(EFM Docs incomplete)

This library came out of my need to achieve a number of file operations that I required to implement tools like: EAT, RG Particle Editor, and others.

The operations you are allowed to do vary by operating system and environment. This library is orgnized to reflect that.
